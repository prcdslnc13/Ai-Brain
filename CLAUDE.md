# Ai-Brain Vault

This Obsidian vault is the persistent **brain** for Claude Code and local LLM runtimes (LMStudio, Ollama).
It is synced across all of your machines via Obsidian Sync. Memory written here on one machine appears
on every other machine within ~30 seconds.

If you are a Claude instance running with this vault as your working directory: your full operating
instructions for using the brain live in your global CLAUDE.md (`~/.claude-personal/CLAUDE.md` or
`~/.claude-work/CLAUDE.md`), which is loaded automatically. The brain is exposed through the
`brain_*` MCP tools — never write to memory by editing files directly; always go through the tools.

Human-readable explanation: see `Brain/README.md`.
Map of memories: see `Brain/_index.md`.
